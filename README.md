# Student Exam Qualification Calculator
## Overview
This console application determines whether a student qualifies to write their final.
* **Test 1:** 30%
* **Test 2:** 50%
* **Assignment 1:** 10%
* **Project:** 10%
A student qualifies if their overall weighted average is **50% or higher**.
